## Initial setup

```sh
 $ git clone https://github.com/DeathSy/cypress-workshop.git
 $ cd cypress-workshop && yarn install
```

## Environment setup (Mac)

✅ git<br/>
✅ nodeJS v10.16 or above<br>
✅ yarn v1.17 or above<br>
✅ VSCODE (recommend)<br>

```sh
$ git --version
$ node --version
$ yarn --version
```
